Important FAQ

What is the Document Object Model (DOM)?

DOM represents the structure of any HTML document as a tree of objects. Each elements of the tree are called nodes.

How do you select an element from the DOM?

To select an element from the DOM a commonly used method is document.getElementById()

What is event delegation in context of the DOM and why is it useful?

Event delegation is a technique in the DOM (Document Object Model) that allows you to manage events efficiently, especially when you're dealing with many elements. It is very useful because of performance efficiency and clear code.

How do you manipulate an element's attributes and styles using the DOM?

There are various ways how element's attributes and styles can be manipulated. Some commonly used methods are, setAttribute() getAttribute() removeAttribute() etc.
